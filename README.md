# Universal AI Sentiment Analysis

AI-powered sentiment analysis system using NLP and Machine Learning.

## Features

- Positive / Negative / Neutral prediction
- Multi-statement analysis
- Confidence score
- Streamlit web app
- NLP preprocessing

## Technologies Used

- Python
- Scikit-learn
- NLTK
- Streamlit
- TF-IDF
- Machine Learning

## Installation

pip install -r requirements.txt

## Run the Project

python train.py

python -m streamlit run app.py

## Dataset

Download IMDb dataset from Kaggle:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
